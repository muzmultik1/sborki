Замечание по СтарКрафту.
========================
1.Работает на всей линейке Виндовс. 

2.На Windows 7, Windows 10 запускать с ddraw.dll ddraw.ini в каталоге. На Windows 95, 98, XP.  Эти файлы удалить инче игра не запустится.

3.На Windows 7, Windows 10 можно запускать в режиме совместимости Режим пониженной цветности 256 цветов.

4.Папках в игре utils\win7colorfix\ фикс цветности на windows 7. sc116-x64.reg – для 64 битной Винды. sc116-x86.reg - Для 32 битной.

5.В папках в игре  utils\server\ файл gate.reg. Пример русского сервера Battle.net для игры через интернет. Если у провайдера заблокирован  порт 6112 то можно создать профиль. Но поиграть не получится.

6.sc_cheats.txt – чит коды.

7.По локальной сети играть через пункт  Local Area Network (UDP). В современных условия пробовал на Windows 7 и Windows 10. Только дать разрешения брандмауэра на доступ в сеть игре.  Можно подключить два старых компьютера напрямую через lan сеть и играть. Если старя сетевая карта до 2010 года  использовать Обжим витой пары кроссовер (crossover). В некоторых роутерах может запрещен доступ wi-fi в lan сеть смотреть настройки.

8.Можно играть с друг другом через  Интернет используя программы для создания виртуальных локальных сетей через интернет. Например radmin vpn. Создаем есть. Даём параметры другу имя и пароль. Соединяемся и играем. Лучше этот способ не использовать. Потому что могут украсть ваши данные с компьютера, Ваш товарищ, который в итоге окажется совсем не товарищ. 

9.Кто хочет играть по Интернету. Можно зайти на сайт https://us.shop.battle.net/ru-ru Скачать клиент Battle.net. Создать учетную запись. Можно через почту Gmail.  Скачать старый бесплатный СтарКрафт  и играть. В самой игре выбрать игру через Battle.net, опять зайти в профиль и  играть.  Люди в онлайн мало, но есть. Если только ваш провайдер опять не заблокировал сайт. Единственный недостаток без входа в онлайн клиент не поиграть.  А так относительно безопасно. 

10.Для чтения справки hlp   редакторе StarEdit.exe. Делал в PowerShell:
irm https://raw.githubusercontent.com/zeljkoavramovic/hlp4win11/main/run.ps1 | iex

11.Реплеи  ложить в папку  maps\ replays.  Карты ложить в папку Maps.  Перемещаться на папку вверх найти  Папку с названием Вверх на один уровень. 

12.При игре по сети появляется пункт Настройка- Сеть. Задержка сети ответа на клик мышки для улучшения соединения. Сейчас не актуально, но возможно будет очень плохая связь. Тогда ставим на очень большая задержка.

13.Если будем создавать диск в Ultra Iso или пробовать запускать игру с записанного cd диска, Игра будет вылетать при прохождении уровня, так как не даётся запись в каталог. Также если допустим мы записали игру на CD диск, а потом скопировали на локальный диск, то смотрим чтоб файлы не стали с атрибутами,  только для чтения. Тогда тоже игра будет вылетать при прохождении уровня. 

14.В меню Настройки – Звук- Субтитры. Включаем если вырезаны звуки. Потому что, когда в сценарии не пройдет звук, то дальше игра не пойдет. И не ясно, что надо будет делать. Можно подумать, что игра зависла. По умолчанию субтитры отключены. 

15.Настройки у меня. Скорость Темп Игры- 3-я точка с права. Звуки Все галочки поставлены, кроме речь юнитов. Субтитры включены. Видео. Коррекция гаммы  - 3-я точка с права
